
# Aplicación Web – Sistema de Gestión de Proyectos

Aplicación web desarrollada para la gestión de proyectos académicos, convocatorias y publicaciones,
consumiendo una API REST desarrollada en Laravel.

El sistema permite a diferentes tipos de usuarios interactuar con el proceso completo
de gestión de proyectos, desde la postulación hasta la certificación.

---

## Tecnologías

- Angular
- TypeScript
- HTML5 / CSS3
- Git

---

## Funcionalidades principales

- Autenticación de usuarios
- Gestión de convocatorias
- Registro y seguimiento de proyectos
- Visualización de fases y estados del proyecto
- Subida y descarga de archivos
- Gestión de observaciones y correcciones
- Visualización y descarga de certificados
- Control de acceso por roles

---

## Arquitectura

- Separación por módulos y servicios
- Consumo de API REST mediante servicios HTTP
- Manejo de estados de navegación
- Organización clara de componentes y servicios

---

## Instalación y ejecución

1. Clonar el repositorio:
```bash
git clone https://github.com/JosephRios7/Linkage-projects-app.git
```

2. Instalar dependencias:
```bash
npm install
```

3. Ejecutar la aplicación:
```bash
ng serve
```

4. Acceder desde el navegador:
```
http://localhost:4200
```
